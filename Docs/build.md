Build
=====

Tools
-----
You need to have installed:
* git
* ninja
* sparse

Toolchain
---------
Clone `https://github.com/primeseven1/lunar-userspace.git`, and run `scripts/gcc.sh`. This script does NOT require root.

Configuration
-------------
Execute `make menuconfig` in the root directory of the project.

Building
--------
Run `make` in the root directory of the project.
